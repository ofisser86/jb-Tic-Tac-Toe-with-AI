Percentage

Write a function get_percentage that takes a real number and represents it as a percentage. So, for example, 0.1 will be equal to 10%, while 1.052 will be 105.2%. Also, add an optional parameter round_digits, which indicates a precision (the number of digits after the point). If it is not specified, round the result to the nearest integer. Here are some examples:

print(get_percentage(0.0123))      # The output is "1%"
print(get_percentage(0.0123, 0))   # The output is "1.0%"
print(get_percentage(0.0123, 1))   # The output is "1.2%"
print(get_percentage(0.0123, 10))  # The output is "1.23%"
print(get_percentage(0.0296, 1))   # The output is "3.0%"

Don't forget to add the percent sign % after the number. Your function should return the result, not print it. The returned percentage should be a string.

Pay attention to the last example: as you can see, a float number there isn't simply truncated to one digit after the point: it is rounded. Use a built-in function round() to do that. It takes up to two arguments, number and ndigits.

Just implement your function, you don't have to handle input or call it in your code.
